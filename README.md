# 🎮 Top-Down Shooter (Unreal Engine C++)

A **Top-Down Shooter game** developed using **Unreal Engine (C++)**, featuring player movement, shooting mechanics, AI enemies, and gameplay systems built with Unreal’s framework.

---

## 🚀 Project Overview

This project demonstrates a **top-down shooter gameplay system** implemented using Unreal Engine with C++. It includes core mechanics such as:

- Player movement & rotation
- Shooting system with projectiles
- Enemy AI behavior
- Collision & damage system
- Animation integration
- Game logic using Behavior Trees & Blackboard

---

## 🛠️ Technologies Used

- Unreal Engine 5
- C++ (Unreal API)
- Behavior Trees (AI)
- Blackboard System
- Unreal Animation System
- Blueprint (optional integrations)

---

## 🎯 Features

### 👤 Player
- Smooth top-down movement
- Mouse-based aiming / directional rotation
- Shooting using projectile system

### 🔫 Combat System
- Projectile-based shooting
- Hit detection and impact effects
- Damage handling

### 🤖 Enemy AI
- AI using Behavior Tree
- Player detection system
- Distance tracking & chasing logic

### 🎞️ Animation
- Character animation blending
- Movement-based animation transitions

## ⚙️ Setup Instructions

1. Clone the repository:
   git clone https://github.com/DURKESH-KUMAR/Top-Down-Shooter-Unreal-CPP.git

2. Open the project:
   - Launch Unreal Engine
   - Click Open Project
   - Select the .uproject file

3. Build the project:
   - Open in Visual Studio
   - Build Solution (Ctrl + Shift + B)

4. Run the game:
   - Click Play inside Unreal Editor
